# 项目介绍

## 1. 这个项目想要干什么？

Claude Code 大面积封禁违规用户，涉及木马程序，且封禁后数据全部丢失、退款困难；同时，Anthropic 也将一系列中转站列入黑名单，此后使用中转 API 调用也会面临困难。

与此相比，OpenAI 推出的 Codex 既有完善的桌面程序界面，对非开发人员十分友好，也有较为宽松的风控机制，并且最新的 GPT5.6-sol/Terra 也比肩 Fable5，是全世界顶尖模型。因此，无论你是否熟悉代码和编程，都应该拥有一个 Codex 应用。这应该作为一个国民级应用来进行推广。

Codex 的功能完善，可以帮你答疑、交作业、翻资料、检索文献、控制远程云主机等等。但是在配置过程中，仍然面临无法访问境外网络、缺少国外手机号、缺乏国际支付银行卡等一系列困难。经过我帮助他人配置过程中的一系列技术探索，我已经总结出了一条适合国内大众的较稳定的保姆级配置方案。从现在开始，以每学期至多 100 元的价格，几乎无限制地调用世界上最为顶级的 agent，对你而言将成为现实。
